скрипт для пакетного создания видео на основе загруженных текстов.
Символ-разделитель для текстов: |
Для основного текста:
<u> подчеркивает слово, перед которым написано (пример: <u>hello)
<l> ставит логотип "logo.gif" в это место. Пример: logo <l>
<b> выделяет слово жирным шрифтом. Пример: <b>bold
<n> переносит последующий текст на новую строку. Пример: строка 1 <n> строка 2
<c> красит слово в цвет, который указан на кнопке "Цвет выделения текста"


Шрифты, логотип находятся в папке for_videos. Можно их просто заменить на те, которые
	найдешь в интернете (но переименуй так, чтобы название было такие-же). Также
	смотри, чтобы расширение файла было таким-же (.gif, .otf и т.д.)
